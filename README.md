# 🛒 Amazon UI Clone

A responsive **Amazon Clone UI** built using **HTML, CSS, and JavaScript**.
This project focuses on replicating the look and feel of the Amazon homepage with modern layout techniques and smooth user interactions.

---

## 🚀 Features

* 🏠 Amazon-like Homepage Layout
* 🔍 Search Bar UI
* 🧭 Navigation Bar with Categories
* 🛍️ Product Cards & Grid Layout
* 🛒 Cart UI (Frontend Only)
* 📱 Fully Responsive Design (Mobile + Tablet + Desktop)
* 🎨 Clean and Modern UI Design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (Flexbox, Grid, Media Queries)
* **JavaScript** – Basic Interactivity
